🚀 启动应用

在终端中运行以下命令启动Web界面：

streamlit run app.py

或使用批处理文件：

run_brush_app.bat
访问应用 (localhost:8501)

🎯 三种工作模式

AI参考解生成

功能：生成最小时间复杂度优先的AI参考解

特点：

  • 题目筛选（难度、标签）
  • Token消耗预估
  • 一键生成最优解
  • 预算管理防超支

模拟面试评估

功能：基于大厂标准的代码面试评估

特点：

  • 7个考核维度评分
  • PASS/FAIL明确判定
  • 结构化反馈报告
  • 面试官模拟评论

刷题训练模式

功能：个性化训练计划（预留）

特点：

  • 进度跟踪
  • 弱点分析
  • 训练统计
  • 待开发功能

🔧 测试要点

请测试以下功能是否正常工作：
  1. 模式切换：在侧边栏切换不同工作模式
  2. 界面响应：主界面是否随模式切换而变化
  3. 表单输入：各个输入框是否正常工作
  4. 按钮功能：预估、生成、评估等按钮
  5. 语言切换：中英文界面切换
  6. 布局检查：界面元素是否排列整齐
